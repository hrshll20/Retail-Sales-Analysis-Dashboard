# Retail Sales Analysis Dashboard

## Project Overview
This project is an interactive Power BI dashboard created for retail sales analysis and customer segmentation.

The dashboard helps analyze:
- Sales performance
- Customer behavior
- Product category performance
- Customer segmentation
- Monthly sales trends

---

## Dashboard Pages

### 1. Sales Analysis Dashboard
- Total Sales
- Total Orders
- Total Customers
- Sales by Category
- Gender-wise Sales
- Monthly Sales Trend

### 2. Customer Analysis Dashboard
- Top Customers
- Age Group Analysis

### 3. Customer Segmentation Dashboard
- Customer clusters
- VIP customers
- Spending behavior
- Quantity analysis
